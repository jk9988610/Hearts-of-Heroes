# Hearts-of-Heroes（三国雄心）

## 在线试玩

合并到 `main` 后自动部署 GitHub Pages：

**https://jk9988610.github.io/Hearts-of-Heroes/**

## 本地运行

```bash
npm install
npm run dev
```

打开 `http://localhost:5173`

## v0.3 功能

- 募兵 / 屯田 / 邻格调兵 / AI 对抗
- 国策面板（广开屯田、坚甲厉兵、急行军）
- 胜负判定（占 6 关键城胜；都城失守或粮尽 5 天负）
- 地图驻军可视化：势力色点 + 兵力数字 + 行军虚线箭头
- 调试：日志从上到下、多选筛选、复制日志、检查版本

### 驻军转移地图表现

| 状态 | 地图显示 |
|------|----------|
| 驻扎 | 格子右下角势力色点 + 底部兵力数字 |
| 行军 | 起点标「出发」；目标格虚线框 +「→N天」；两格间箭头 |
| 战斗 | 左上角「战N」+ 兵力数字 |

### 调试工具栏

- **打印详情**：结构化快照（含军队一览）
- **复制日志**：复制当前筛选可见日志
- **检查版本**：对比本地与 `version.json`（Pages 部署后有效）
