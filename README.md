# Hearts-of-Heroes（三国雄心）

## Sprint 0 本地测试

### 你需要做的（一次性环境）

1. 安装 [Node.js](https://nodejs.org/) 18+（推荐 LTS）
2. 克隆本仓库并进入目录
3. 安装依赖：

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

终端会输出本地地址，一般为 `http://localhost:5173`。用 Chrome / Edge 打开即可。

### 测试清单

| 项目 | 操作 | 预期结果 |
|------|------|----------|
| 地图 | 打开页面 | 6×6 色块地图，12 座关键城有名称 |
| 时间 | 等待约 5 秒 | 顶部「第 N 天」递增；日志每 5 天打印 `gameTick` |
| 暂停 | 按空格或点「暂停」 | 天数停止增长，日志显示「游戏暂停」 |
| 变速 | 点 ×2 / ×5 | 天数增速变快 |
| 选地 | 点击地图格子 | 日志显示地块名、地形、归属、邻接数 |
| 存档 | 点「手动存档」，刷新页面 | 天数与地图归属保持不变 |
| 读档 | 改天数后点「读档」 | 恢复上次存档天数 |
| 新游戏 | 点「新游戏」确认 | 天数归零，势力归属重置 |

### 浏览器调试

- 按 `F12` 打开开发者工具 → **Console** 可看完整日志
- **Application → IndexedDB → sanguo-save → saves** 可查看存档 JSON

### 生产构建预览（可选）

```bash
npm run build
npm run preview
```

访问 `http://localhost:4173` 验证打包后是否正常。
