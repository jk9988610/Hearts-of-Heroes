# 三国雄心 · v1.0 大版本路线图

> 当前基线：**v0.6.0**（Sprint A 体验打磨完成）  
> **最新详细计划**：[`UPDATE_PLAN_V07.md`](./UPDATE_PLAN_V07.md) — 编制单位、地图分层、大地图 UI、v0.7→v1.0 分 Phase

## 愿景摘要（v0.7+）

- **编制**：百人队（组成）→ 千人队（**自动番号**，无队长）→ 将军队（**≤10 队** + 将军）→ 集团军（**手动**组建 + 元帅）。
- **将领栏**：默认隐藏；「新编军队」+ 待命按钮左向增长、排满上移；长按编入、单击左上详情弹窗（将军任命 + 番号列表 + 编队详情）。
- **地图分层**：军情（默认）/ 地形 / 资源（**名称+数字**）；底栏右侧单选。
- **UI**：大地图主屏；顶栏收纳调试等；弹窗叠在地图之上；底栏 = 小地图（**共用 map 实例**）| 将领栏 | 图层。

详细交互与 HOI4 取舍见 [`UPDATE_PLAN_V07.md`](./UPDATE_PLAN_V07.md)。

## v0.5 已完成要点

| 能力 | 说明 |
|------|------|
| 小时 Tick | 1 真实秒 = 1 游戏日 = 24 游戏小时；行军 48h、战斗 72h |
| 玩家主权 | `playerFaction` 完全由玩家操作，AI 跳过该势力 |

## v1.0 目标

**主题：可发布的完整单机体验 + 可选联机存档 + 移动端**

预计拆为 4 个 Sprint，按依赖顺序推进。

### Sprint A — 体验打磨（P0）✅ v0.6.0

1. **开局势力选择弹窗** ✅
   - 新游戏前模态选择魏/蜀/吴，展示首都与初始兵力
   - 取代工具栏「下次新游戏生效」的隐式流程

2. **战斗数字动画** ✅
   - 战斗格兵力变化 tween（攻损/守损）
   - 攻克/守住时短暂高亮与震动反馈

3. **地图缩放** ✅
   - 双指 pinch / 滚轮缩放（`map-viewport` + Canvas 变换）
   - 缩放时保持选中格与视口中心

4. **UI 信息密度** ✅
   - 顶栏显示「第 N 天 HH 时」
   - 侧栏分块：地块 / 军队 / 国策 / 事件摘要

### Sprint B — 武将与国策（P1）→ 已并入 `UPDATE_PLAN_V07` Phase 3 / v0.9

1. **武将系统（最小可用）** — 见 v0.9
   - 首都绑定 1 名主将：`heroes.json` 攻击/防御加成进战斗公式
   - 面板显示驻地将领；募兵时主将所在格加成

2. **国策树扩展**
   - 3 条分支各 2～3 级（屯田、行军、防御）
   - 互斥与前置条件；国策面板改为树状

3. **粮尽与溃散**
   - 连续缺粮 N 天：兵力按比例流失
   - 与现有 `starvingDays` 字段打通

### Sprint C — AI 战略层（P1）→ 已并入 `UPDATE_PLAN_V07` Phase 4 / v1.0

1. **分层 AI（仿 HOI4 深化）**
   - **战略层**（每游戏日 1 次）：选定扩张方向、优先目标城
   - **战术层**（视窗内每 1h）：进攻/防守/募兵
   - **精简层**（视窗外每 6h）：仅经济与募兵（v0.5 已有）

2. **多方博弈**
   - 弱势力结盟倾向（暂不实现玩家外交 UI）
   - 避免三方可预测循环互殴

3. **难度档位**
   - 简单：AI 粮食 +20%、进攻犹豫
   - 困难：AI 决策间隔缩短、优先夺关键城

### Sprint D — 平台与存档（P2）→ 已并入 `UPDATE_PLAN_V07` Phase 4 / v1.0

1. **Supabase 云存档**
   - 匿名登录 + `saves` 表；本地 IndexedDB 为主、云端异步同步
   - 冲突策略：时间戳较新者胜，合并前弹窗确认

2. **Capacitor Android APK**
   - 全屏、返回键暂停、触控优化
   - CI 产出 debug APK artifact

3. **PWA / 离线**
   - `manifest.json` + Service Worker 缓存静态资源与配置

## 验收标准（v1.0 Release）

- [ ] 完整一局 30 分钟内可分出胜负，无卡死

- [ ] 新档/读档/云同步在 Pages 与 APK 上一致
- [ ] 玩家势力全程无 AI 越权操作
- [ ] 视窗外 AI 不导致帧率明显下降（60fps 目标）
